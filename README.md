## 📊 Overview

This project analyzes hospital admission data and presents insights using an interactive dashboard built with Streamlit.

## 🚀 Features

* Data Cleaning & Analysis using Pandas
* Interactive Dashboard with Streamlit
* KPI Metrics (Total Records)
* Dynamic Visualizations using Plotly

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Plotly
* Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 💡 Insights

* Identified patterns in hospital admissions
* Visualized data trends
* Built interactive filters for analysis
